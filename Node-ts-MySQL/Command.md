# MySQL 명령어

### Fetching
```js

SHOW DATABASES - db 다 보여주기
SHOW TABLES - 테이블 다 보여주기
SELECT * FROM [TABLE NAME] - 특정 테이블의 데이터 가져오기
SELECT * FROM [TABLE NAME] ORDER BY [FILED NAME] DESC|ASC - 가져올때 sort
SELECT * FROM [TABLE NAME] WHERE [FILED NAME] LINK 'aa%' - 가져올때 aa로 시작하는거만
SELECT * FROM [TABLE NAME] WHERE [FILED NAME] LINK '%aa' - 가져올때 aa로 끝나는거만
SHOW FULL COLUMNS FROM [TABLE NAME] - 모든 Columns 보여주기

```
